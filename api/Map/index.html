size​

Map.size: number

Returns the number of key/value pairs in the Map object.

Functions​

new​

</>
Map.new(
arg?: table | Array--

The initial value to use when constructing

) → Map

Creates a new Map object.

  • If a table is supplied, this will "deep copy" the table into this Map.
  • Passing an Array with key-value pairs is also acceptable.

clear​

</>
Map:clear() → ()

Removes all key-value pairs from the Map object.

delete​

</>
Map:delete(
key: any--

The key of the element to remove from the Map object.

) → boolean

Returns true if an element in the Map object existed and has been removed, or false if the element does not exist. map.has(key) will return false afterwards.

get​

</>
Map:get(
key: any--

The key of the element to return from the Map object.

) → any?

Returns the value associated to the passed key, or nil if there is none.

</>
Map:has(
key: any--

The key of the element to test for presence in the Map object.

) → boolean

Returns a boolean indicating whether a value has been associated with the passed key in the Map object or not.

set​

forEach​

</>
Map:forEach(
callbackFn: fun(value: any,key: any,map: any),--

Function to execute for each entry in the map. It takes the following arguments:

thisArg?: any--

Value to use as self when excuting callback.

) → ()

Calls callbackFn once for each key-value pair present in the Map object, in insertion order. If thisArg parameter is provided to forEach, it will be used as the self value for each callback.

  • value (optional) -- Value of each iteration.
  • key (optional) -- Key of each iteration.
  • map (optional) -- The Map being iterated.
